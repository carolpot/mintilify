The **Lusha MCP server** implements the [Model Context Protocol](https://modelcontextprotocol.io/) — a standard that lets AI assistants and agents call external tools directly. Once configured, any MCP-compatible client (Claude, Cursor, Windsurf, and others) can enrich contacts, search for prospects, and retrieve company data using natural language — without you writing a single API call.

## What you can do with Lusha MCP

- Ask an AI assistant to "enrich this contact" and get back verified emails and phone numbers
- Instruct an agent to "find 20 VP of Engineering contacts at Series B companies in New York"
- Trigger a prospect list build from inside a Claude or Cursor session
- Chain Lusha tools with other MCP tools (CRM writes, email sends, Slack notifications) in a single agent workflow

## How it works

```
Your AI client (Claude, Cursor…)
        ↓  MCP tool call
Lusha MCP server
        ↓  REST API call
Lusha API  →  enriched data
        ↑
returned to your AI client
```

The MCP server translates natural-language tool calls into authenticated Lusha API requests and returns structured results that the AI can reason over or pass downstream.
