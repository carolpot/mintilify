Retrieve the list of valid industry values you can use to filter companies in `POST /prospecting/company/search`. Use the exact strings returned by this endpoint to ensure your industry filters are recognized correctly.

**Endpoint:** `GET https://api.lusha.com/prospecting/filters/companies/industries_labels`

## Example request

```bash
curl --request GET \
  --url https://api.lusha.com/prospecting/filters/companies/industries_labels \
  --header 'api_key: YOUR_API_KEY'
```

## Example response

```json
{
  "industries": [
    "Technology",
    "Financial Services",
    "Healthcare",
    "Manufacturing",
    "Retail",
    "Education",
    "Real Estate",
    "Media & Entertainment"
  ]
}
```

## Usage

Pass one or more values from the `industries` array into the `industries` filter field of your company search request:

```json
{
  "industries": ["Technology", "Financial Services"]
}
```

<Tip>
  Cache the response from this endpoint. Industry values change infrequently.
</Tip>
