Retrieve the list of valid seniority level values for filtering contacts in `POST /prospecting/contact/search`. Seniority levels range from entry-level individual contributors to C-suite executives. Always use values returned by this endpoint to ensure your search filters are recognized correctly.

**Endpoint:** `GET https://api.lusha.com/prospecting/filters/contacts/seniority`

## Example request

```bash
curl --request GET \
  --url https://api.lusha.com/prospecting/filters/contacts/seniority \
  --header 'api_key: YOUR_API_KEY'
```

## Example response

```json
{
  "seniority": [
    "Entry",
    "Manager",
    "Director",
    "VP",
    "C-Level",
    "Partner",
    "Owner",
    "Founder"
  ]
}
```

## Usage

Pass one or more values from the `seniority` array into the `seniority` filter field of your contact search request:

```json
{
  "seniority": ["Director", "VP", "C-Level"]
}
```

<Tip>
  Cache the response from this endpoint. Seniority values rarely change and do not need to be fetched on every search run.
</Tip>
