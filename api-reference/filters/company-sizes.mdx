Retrieve the list of available company size range values for filtering companies by headcount in `POST /prospecting/company/search`. Each value represents a headcount band. Use the exact strings returned by this endpoint in your search request.

**Endpoint:** `GET https://api.lusha.com/prospecting/filters/companies/sizes`

## Example request

```bash
curl --request GET \
  --url https://api.lusha.com/prospecting/filters/companies/sizes \
  --header 'api_key: YOUR_API_KEY'
```

## Example response

```json
{
  "sizes": [
    "1-10",
    "11-50",
    "51-200",
    "201-500",
    "501-1000",
    "1001-5000",
    "5001-10000",
    "10001+"
  ]
}
```

## Usage

Pass one or more size range strings from the `sizes` array into the `companySizes` filter field of your company search request:

```json
{
  "companySizes": ["51-200", "201-500"]
}
```

<Tip>
  Cache the response from this endpoint. Size ranges are stable and do not change frequently.
</Tip>
