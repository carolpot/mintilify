The Lusha API gives you programmatic access to one of the world's largest B2B contact and company databases. Use it to enrich existing CRM records with verified emails and phone numbers, search for new prospects that match your ideal customer profile, subscribe to real-time job-change and company-event signals, and discover lookalike contacts or companies similar to your best customers.

## How the Lusha API works

<Steps>
  <Step title="Get your API key">
    Sign up at [lusha.com](https://www.lusha.com/signup/) and copy your API key from the API settings page.
  </Step>
  <Step title="Authenticate requests">
    Pass your key in the `api_key` header on every request to `https://api.lusha.com`.
  </Step>
  <Step title="Enrich or prospect">
    Call the Enrichment API to fill in missing data on known contacts, or use the Prospecting API to discover new ones.
  </Step>
  <Step title="Subscribe to signals">
    Set up Webhooks to receive instant notifications when tracked contacts change jobs or companies reach growth milestones.
  </Step>
</Steps>

<Note>
  All Lusha API requests are credit-based. Check your current credit balance and usage at any time with the [Account Usage](/account/usage) endpoint.
</Note>
