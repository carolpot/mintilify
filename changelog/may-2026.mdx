## May 12, 2026

### New: MCP server (beta)

The Lusha MCP server is now in public beta. Connect Lusha data to Claude, Cursor, Windsurf, and any other MCP-compatible client. See the [MCP documentation](/mcp/overview) to get started.

### Fixed: API Reference endpoint badges

All API Reference pages now correctly display the HTTP method badge and interactive playground. Previously, enrichment, signal, lookalike, and webhook pages were missing the method indicator.

---

## May 1, 2026

### Updated: Bulk company endpoint path

The bulk company enrichment endpoint is now documented at its correct path, `POST /bulk/company/v2`. The OpenAPI spec previously listed this operation under `POST /v2/company`.

<Note>
  This was a documentation fix only — the API endpoint itself did not change.
</Note>
