<Accordion title="What is MCP?">
  The Model Context Protocol (MCP) is an open standard that lets AI assistants and agents call external tools in a structured, consistent way. It means you can ask Claude or Cursor to "enrich this contact" and have it call the Lusha API automatically — no manual API calls required.
</Accordion>

<Accordion title="Which AI clients support MCP?">
  MCP is supported by Claude Desktop, Cursor, Windsurf, and a growing list of other AI development tools. See the [MCP setup guide](/mcp/setup) for client-specific instructions.
</Accordion>

<Accordion title="Do I need a separate Lusha plan for MCP?">
  No — MCP uses the same API key and credits as the REST API. Any Lusha plan that includes API access can use the MCP server.
</Accordion>

<Accordion title="Is MCP billed differently from the REST API?">
  No. MCP tool calls map 1:1 to Lusha API requests. Credits are deducted the same way — per successful enrichment result.
</Accordion>
